# Performance & Consistency Analysis: Lewis Hamilton

**OVERVIEW**
This project analyzes the career performance of Lewis Hamilton, one of the most successful drivers in Formula 1 history.
The dashboard explores key metrics such as rce wins, podium finishes, consistency, and race outcomes over time using Power BI

**OBJECTIVES**
-Analyze race wins and podium trends
-Evaluate consistency across seasons
-Identify peak peeformance periods
-Examine race outcomes and reliability

**KEY INSIGHTS**
-Lewis Hamilton demonstrates exceptional consistency with a high podium rate
-Peak performance observed during dominant championship seasons
-Strong correlation between race finishes and championship wins
-Low DNF rate contributes significantly to overall success

**TOOLS USED**
-Power BI
-Power Query (Data Cleaning)
-DAX(Measures and KPIs)
